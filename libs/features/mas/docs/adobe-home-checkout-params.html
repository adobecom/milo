M@S - Checkout Parameters Use Case

Checkout Parameters allows adding valid UCv3 request parameters to the checkout link.

1. Invalid Content

Checkout Parameters value: inapp%3A%2F%2Fahome%3Fworkflow=cme-1&targetURL=%7Bsegmentation%3Fcli%3D%24%7Bcli%7D&co=%24%7Bco%7D&lang=%24%7Blang%7D&pa=ccsn_direct_individual&ot=trial%7D&ctxRtUrl=%7BctxRtUrl%7D
Result Checkout CTA is a valid UCv3 link: https://commerce.adobe.com/store/segmentation?cli=adobe_com&ctx=fp&co=%24%7Bco%7D&lang=en&pa=ccsn_direct_individual&ot=TRIAL&ctxRtUrl=%7BctxRtUrl%7D&ms=COM&cs=INDIVIDUAL.

2. Valid Content

Checkout Parameters value: ctx=if&mv=other&mv2=ahome
Result Checkout CTA is a valid UCv3 link: https://commerce.adobe.com/store/segmentation?cli=adobe_com&ctx=fp&co=US&lang=en&ms=COM&ot=BASE&cs=INDIVIDUAL&pa=ccsn_direct_individual.
List of valid parameters: https://wiki.corp.adobe.com/display/businessservices/UCv3+Link+Creation+Guide
